A Linktree style personal page built with React/Vite that lists personal and professional links while introducing a new feature: an AI-assisted metadata search engine. The search engine helps visitors quickly find specific information they are looking for by highlighting the most relevant links and displaying short contextual explanations describing what was found.

Live Demo: https://selam-linktree.netlify.app/ 

Running Locally
- Clone the repository `git clone <repo-url>`
- Navigate into the project `cd my-linktree`
- Install dependencies `npm install`
- Start with `npm run dev`

Feature: AI-Assisted Metadata Smart Search

The main feature of this project is an AI-assisted metadata smart search engine designed specifically for Linktree pages. Instead of visitors manually opening every link to search for information, they can search for skills, projects, experience, or topics directly from the page. The system then highlights the links where the requested information can be found and displays short contextual explanations describing what was matched. For example, searching “Kubernetes experience” highlights the Resume link and displays a short explanation about Kubernetes deployment experience during AIEA research work. The goal of the feature is to improve navigation and discovery while still preserving the simplicity of a traditional Linktree.
